🎓 Student Registration Portal

A simple and responsive **Student Registration Portal** developed using **PHP, MySQL, Bootstrap, HTML, and CSS**. This project allows users to register students, store their details in a MySQL database, and manage student records through CRUD (Create, Read, Update, Delete) operations.

---

🚀 Features

* Student Registration Form
* Save Student Details to MySQL Database
* View All Registered Students
* View Student Details
* Edit Student Information
* Delete Student Records
* Search Students
* Email Validation
* Mobile Number Validation
* Duplicate Email Check
* Responsive User Interface using Bootstrap

---

 🛠️ Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap 5
* XAMPP

## ⚙️ Setup Instructions

1. Install XAMPP.
2. Copy the project folder into the `htdocs` directory.
3. Start Apache and MySQL from the XAMPP Control Panel.
4. Create a database named `student_registration`.
5. Import the `student_registration.sql` file using phpMyAdmin.
6. Update the database credentials in `database.php` if necessary.
7. Open your browser and visit:

```
http://localhost/student-registration/
```
